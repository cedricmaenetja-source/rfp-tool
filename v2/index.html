Requirements

Each Excel file you upload should contain the following columns:

  • area — the functional area or category
  • requirement — the requirement description
  • priority — Must-Have, Should-Have, or Could-Have
  • system_part — the system module or component
  • response — the vendor's response value
  • notes — any additional comments

Column headers are case-insensitive. The filename will be used as the vendor name.
