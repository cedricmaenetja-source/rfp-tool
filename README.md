# rfp-tool

## setup
### Prerequisite
- Node.js 

### Steps
- Clone repo.

- Install vercel
```yaml
npm install -g vercel
```

Check version to confirm its installed
```yaml
vercel --version 
```

- Install supabase
```yaml
scoop install supabase
```

Check version to confirm its installed
```yaml
supabase --version 
```
- Install supabase cli
```yaml
npm install @supabase/supabase-js
```
